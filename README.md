# 车牌检索app
练手项目，用于在已知车牌信息数据库中以车牌为索引查找全部信息，显示以及完成后续处理。
# APP功能
- 输入车牌号，查询数据库中是否有该车牌信息，若有，显示全部信息，语音播报车位号；若无，提示未找到。
- 点击车牌信息，可进行修改或删除操作。
- 点击添加信息，可添加新的车牌信息
- 点击手机号信息，调用系统拨号，拨打选中号码。。
# 车牌信息结构
- 数据库结构  
  - ParkingInfo("28#", "杨彦芳", "28#", "京N6W722", "13301156775"));
- 车牌信息显示结构  
    - 车位号：28#
    - 姓名：杨彦芳
    - 车牌：京N6W722
    - 手机号：13301156775
